# Emotion Diary Frontend 📖

![React](https://img.shields.io/badge/React-19.0.0-blue)

**Дневник эмоций** - современное веб-приложение для отслеживания настроения с аналитикой и интуитивным интерфейсом.

## ✨ Особенности

- 📅 **Ежедневное отслеживание** эмоционального состояния
- 🎨 **Современный дизайн** с зелёной цветовой схемой
- 📊 **Визуальная статистика** и аналитика настроения
- 📱 **Адаптивный интерфейс** для всех устройств
- ⚡ **Быстрая работа** благодаря Vite

## 🛠 Технологии

- **Frontend:** React 19, Vite
- **Стили:** CSS3, React Bootstrap
- **Графика:** Recharts для визуализации
- **Календарь:** React Calendar
- **Маршрутизация:** React Router DOM
